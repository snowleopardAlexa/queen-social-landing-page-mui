## Social Landing Page with Material-UI

This is Social Landing Page about Queen band. It is made with Material-UI ReactJS. 

The project covers the most popular Material-UI components:

- Theme
- Navbar
- Responsive Design
- Props
- Grid
- Sidebar
- Card
- Tooltip
- Modal
- Form
- Snackbar
- Right Menu 

### Dependencies

- @material-ui/core
- @material-ui/icons
- @material-ui/styles
